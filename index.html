About IBNIZ

IBNIZ is a minimalist virtual machine and esoteric programming language for creating audiovisual art and demos.

In IBNIZ, you write short programs that generate both graphics and sound. The language uses a stack-based approach where operations manipulate coordinates and time to create patterns, animations, and audio.
